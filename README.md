# my-hello-agent

A minimal Python application managed with `uv`.

## Requirements

- `uv`
- Python `3.14`

## Getting started

```bash
uv sync
uv run my-hello-agent
```

## Project layout

- `pyproject.toml`: project metadata and script entrypoint
- `src/my_hello_agent/__init__.py`: application entrypoint
- `uv.lock`: dependency lockfile generated by `uv sync`
